# `@tookey-io/libtss`

Helper library with realization for Threshold Signatures and common code for other clients.

# Rust usage
Add `tookey-libtss = { version = "1.0.3", default-features = false, features = ["gmp"] }` to Cargo.toml

# NodeJs usage
Run `yarn add `
